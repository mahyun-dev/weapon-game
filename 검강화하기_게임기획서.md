# 검 강화하기 - 게임 기획서

## 📋 게임 개요

**장르**: 방치형 클리커 RPG  
**플랫폼**: 웹 브라우저  
**타겟**: 캐주얼 게이머  
**핵심 루프**: 클릭 → 골드 획득 → 검 강화 → 더 강한 검 획득 → 반복

---

## 🎮 핵심 게임 시스템

### 1. 기본 시스템

#### 1.1 클릭 시스템
- 화면 중앙의 검을 클릭하여 골드 획득
- 클릭당 기본 골드: 현재 강화 수치에 따라 증가
- 자동 클릭 시스템 (장비 착용 시)

#### 1.2 골드 시스템
- 기본 화폐 단위
- 검 강화, 상점 구매, 대장간 조합에 사용
- 표시 단위: 원 (₩)

### 2.2 강화 실패 시스템

- **강화 실패**: 성공률에 따라 실패 가능
- **실패 패널티**: 
  - +0~+9: 강화 단계 유지, 골드만 소모
  - +10~+19: 강화 단계 -1 하락
  - +20~+30: 강화 단계 -2 하락
- **방지권 사용**: 실패 시 강화 단계 하락 방지 (골드는 소모됨)

### 2.3 강화 보호 시스템

## 🎨 UI/UX 설계

### 7.1 메인 화면 구성

```
┌─────────────────────────────────────┐
│  [칭호] 검 강화하기        [설정] │
│                                     │
│      현재 골드: 1,000,000원         │
│                                     │
│         [검 이미지]                 │
│       +15 시간의 검                 │
│                                     │
│    클릭당: 50,000원                 │
│    성공률: 65%                      │
│                                     │
│  [아이템창기] [상점가기] [나가다]   │
│                                     │
│  방지권: 0         돈:              │
└─────────────────────────────────────┘
```

### 7.2 하단 메뉴

- **아이템창기**: 장비 장착 및 보유 아이템 확인
- **상점가기**: 워프권, 방지권, 특별 아이템 구매
- **조합소가기**: 대장간에서 재료 조합
- **판매하다**: 현재 검 판매
- **나가다**: 메인 화면으로 돌아가기

### 7.3 강화 효과

- **성공 시**: 
  - 검이 빛나는 애니메이션
  - "+1" 텍스트 상승 효과
  - 성공 사운드
  
- **실패 시**:
  - 검이 깨지는 듯한 효과
  - "강화 실패" 텍스트
  - 실패 사운드
  - 방지권 사용 여부 팝업

## 🔄 밸런스 조정 요소

### 9.1 골드 밸런스

- 초반(+0~+10): 빠른 진행을 위한 낮은 가격
- 중반(+11~+20): 점진적인 난이도 상승
- 후반(+21~+30): 장기 플레이를 위한 높은 난이도

### 9.2 성공률 밸런스

- +0~+9: 100%~85% (튜토리얼 단계)
- +10~+19: 80%~55% (중간 난이도)
- +20~+30: 55%~30% (하드코어)
