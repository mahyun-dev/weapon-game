# 검 강화하기 - Idle Clicker RPG

웹 브라우저에서 즐길 수 있는 방치형 클리커 RPG 게임입니다.

[![Live Demo](https://img.shields.io/badge/Play-Live%20Demo-brightgreen)](https://mahyun-dev.github.io/weapon-game/)
![Platform](https://img.shields.io/badge/Platform-Web%20Browser-brightgreen)
![Language](https://img.shields.io/badge/Language-JavaScript-yellow)
![Language](https://img.shields.io/badge/Language-Css-purple)
![Language](https://img.shields.io/badge/Language-Html-orange)

## 🎮 게임 소개

- **장르**: 방치형 클리커 RPG
- **플랫폼**: 웹 브라우저
- **타겟**: 캐주얼 게이머
- **핵심 루프**: 클릭 → 골드 획득 → 검 강화 → 반복

## ✨ 주요 특징

### 🔨 강화 시스템
- +0부터 +30까지 총 31단계 강화
- 성공/실패 확률 기반 도박성 요소
- 크리티컬 강화 (5% 확률로 +2 상승)
- 천장 시스템 (연속 실패 시 100% 성공 보장)

### 🛡️ 방지 시스템
- 방지권으로 강화 실패 방지
- 천장 시스템으로 과도한 실패 방지
- 무과금 플레이어 배려

### 🏪 상점 시스템
- 워프권: 클릭 배율 증가
- 방지권: 강화 실패 보호
- 특별 아이템: 영구 효과 부여

### 🔨 대장간 시스템
- 재료 조합으로 아이템 제작
- 강화 성공 시 재료 드롭
- 전략적 자원 관리

### 🎒 장비 시스템
- 10개 슬롯의 다양한 장비
- 랜덤 상자로 획득
- 효과 중첩 가능

### 📊 업적 시스템
- 18개 다양한 업적
- 달성 시 골드/아이템 보상
- 숨겨진 업적 포함

### 🏆 칭호 시스템
- 10개 특별 칭호
- 영구 효과 부여
- 희귀도에 따른 시각적 효과

## 🛠️ 기술 스택

### 프론트엔드
- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인, 애니메이션
- **Vanilla JavaScript**: 게임 로직, 상태 관리
- **LocalStorage**: 클라이언트 측 데이터 저장

### 데이터 관리
- **JSON 파일**: 게임 데이터 관리
- **모듈화**: 각 시스템별 분리된 데이터 파일
